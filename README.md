# eject-usb

🧩 USBメモリを安全に強制取り外しするポータブルツール（Windows用）

## 🔧 特徴

- 接続中のUSBドライブを一覧表示
- 番号で選んで取り外し
- `handle64.exe` を内蔵し、掴んでいるプロセスを検出・強制終了
- 完全ポータブル：`.exe` 単体で実行可
- PowerShell・外部ツールの事前準備不要

## ✅ 使い方

1. `eject-usb.exe` をダブルクリック
2. 接続されているUSBの一覧が表示されます
3. 番号を入力すると、そのドライブを強制取り外します
